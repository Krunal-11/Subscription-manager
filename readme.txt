this is a subscription manager application that helps in managing the subscriptions. It can be accessed by going to (http://localhost:8080/home.html)
The application has 2 views user view and admin view 
login with username and password as 'root' for all users
it has add subscription, delete subscription, edit subscription, and view subscriptions. 

The admin panel has insights based on the records inserted in the table.
The add, edit , delete and view functionality are all available on this url (http://localhost:8080/admin.html) 

The user panel has information about particular user where they can check their subscriptions information and status
